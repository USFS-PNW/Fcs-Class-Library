This project has third-party references.

There are 3 legacy Devart 6.8.0 references: Devart.Data.dll, Devart.Data.Oracle.dll, and Devart.Data.Oracle.Entity.dll. Devart is licensed software so these .dll's are not stored
in GitHub where they are publicly downloadable. These libraries may be available upon request 
from the repository owner when proof of a DEVART license is supplied. This project is configured 
to use libraries from the ./lib folder.


EntityFramework v5.0 is a Microsoft library and can be installed using the NuGet Package Manager. From
the Package Manager Console, type:
install-package entityframework -version 5.0.0.0
